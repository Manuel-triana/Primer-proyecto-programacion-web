Pagina con diferentes tipos de input

Esta pagina es con el fin de conocer mas acerca de los tipos de input que hay y para que sirve cada uno.

Input de tipo texto : Permite ingresar una cadena de texto. Ideal para nombres, direcciones, etc.

Input de tipo numero : Permite ingresar solo números y puede tener restricciones como valores mínimo y máximo.

Input de tipo Date : Permite seleccionar una fecha usando un selector de fecha.

Input de tipo rango : Ideal para seleccionar una sola opción de un conjunto predefinido.
